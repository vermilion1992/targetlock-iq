# 5-minute demo script — TargetLock IQ v2 RC1

**Audience:** Supervisor or project lead  
**Mode:** Simple, then flash Advanced Branch program

1. **Open** `/targetlock` — note version label `TargetLock IQ v2 RC1`.
2. **KPI strip** — status, projected miss, plan offset (sample hole already loaded).
3. **Action plan** — advisory wording; not a rig instruction.
4. **Add one survey** — show aim pre-fill and KPI update.
5. **Export PDF** — handover for shift record.
6. **Advanced → Branch program** (optional 60 s) — program map, one daughter, kickoff planner headline.
7. **Close** — “Data stays on your browser; export JSON package for backup; pilot feedback welcome.”

**Do not say:** “The app tells the driller what to do.”  
**Do say:** “It surfaces projection, tolerance, and recovery options for team discussion.”
