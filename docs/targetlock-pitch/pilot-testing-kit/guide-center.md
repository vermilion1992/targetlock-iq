# Guide Center — in-app walkthroughs

**In app:** Top bar → **Guide** → **Guide Center**

TargetLock IQ ships three institutional guides. Your hole data is **not** changed when you start a guide. Use **Load demo** on a step only when you need synthetic data (restored on **Exit guide**).

| Guide | Steps | Time | Use when |
|-------|-------|------|----------|
| **Quick orientation** | 6 | ~5 min | First visit; layout and modes |
| **Standard hole workflow** | 20 | ~25 min | Plan → surveys → action plan → export |
| **Branch / daughter hole workflow** | 12 | ~20 min | Mother hole, kickoff, approval, branch PDF |

## Quick orientation (6 steps)

1. What TargetLock IQ is for  
2. Sidebar = setup and data  
3. Main workspace = decisions  
4. Simple mode = field-ready summary  
5. Advanced mode = technical review  
6. Scenario Lab = safe testing data  

## Standard hole workflow (20 steps)

1. Welcome  
2. Simple vs Advanced  
3. Hole details  
4. Upload a hole plan  
5. Upload survey results  
6. Read the four key KPIs *(optional demo: gradual drift)*  
7. Understand the Action Plan  
8. Add a new survey manually  
9. Use “Fill from action plan”  
10. Check Plan / Section / Deviation charts  
11. Open Advanced mode  
12. Review Steering Feasibility *(optional demo: motor / Navi review)*  
13. Review QA/QC flags  
14. Check Math Reference  
15. Set Survey Tool Profile  
16. Adjust Plan Corridor / Target setup  
17. Use Scenario Lab  
18. Export handover PDF *(optional demo: recoverable correction)* — page 1 decision summary + technical detail appendix when relevant  
19. Export / import full hole package  
20. Known limitations and validation reminder  

## Branch / daughter hole workflow (12 steps)

1. Mother/daughter program concept  
2. Load Branch Program scenario *(optional demo: P2 kickoff comparison)*  
3. Mother hole actual path  
4. Add / select daughter targets  
5. Review kickoff options  
6. Understand DLS and separation  
7. Save daughter draft  
8. Approve daughter plan  
9. Switch active daughter *(optional demo: three targets)*  
10. Import daughter survey results  
11. Export branch plan PDF — branded branch plan with mother/daughter trajectory visual  
12. Operational disclaimer  

## Controls during a guide

| Control | Action |
|---------|--------|
| **Previous / Next** | Move between steps |
| **Open [tab]** | Switches to Advanced and opens the relevant tab |
| **Load demo** | Loads a built-in or branch scenario (confirm dialog) |
| **Restart** | Restores your pre-guide data and returns to step 1 |
| **Exit guide** | Ends guide and restores pre-guide state |
| **Guide center** | Reopens flow picker without ending the guide |
