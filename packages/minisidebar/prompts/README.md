# TailwindAdmin Pro Prompts Explorer

Welcome to the Prompts Explorer! This library contains pre-written AI prompts optimized for Claude, GPT-4, and Gemini to generate, update, and theme components in the TailwindAdmin Pro architecture.

Click on a folder to expand it
Then click on a file to preview its content

────────────────────────────

## How to Use

1. **Find a Prompt:** Navigate the folders below to find the task you want to perform.
2. **Fill Placeholders:** Open the `.prompt.md` file and look for variables like `{{MODE}}` or `{{PRIMARY_COLOR}}`.
3. **Copy to AI:** Copy the entire prompt text, fill in your specific variables, and paste it into your AI agent (Cursor, ChatGPT, Claude, etc.).
4. **Execute:** The AI will output code that perfectly matches the TailwindAdmin Pro DNA.

────────────────────────────

## Available Categories

**apps** - Application & standalone feature prompts
**branding** - Logo, branding & SVG updates
**components** - Reusable UI component creation
**dashboard** - Dashboard layout & chart setup
**forms** - Form validation & component prompts
**language** - i18n & multi-language localization
**navigation** - Menu, sidebar & routing changes
**theming** - Global light/dark modes & OKLCH color adjustments
